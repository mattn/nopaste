# nopaste

## Usage

```
Usage of nopaste.exe:
  -addr string
    	server address (default ":8989")
  -data string
    	data directory (default "data")
```

## Installation

```
$ go get github.com/mattn/nopaste
```

## Development

```
$ go build
```
